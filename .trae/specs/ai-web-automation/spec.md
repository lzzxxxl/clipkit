# AI网页全自动批量处理工具 - 产品需求文档

## Overview
- **Summary**: 一个整合网页自动化操作与剪贴板保存功能的一体化工具，用于批量向网页版AI（如ChatGPT、DeepSeek、Grok等）提交问题并自动保存回复结果。
- **Purpose**: 解决用户频繁向多个网页版AI批量提交标题/问题时的效率低、重复劳动、易遗漏等问题，实现全流程自动化处理。
- **Target Users**: 需要批量向网页版AI提交问题、获取回复并保存结果，追求高效、无人值守操作的用户。

## Goals
- 实现从标题读取、批量提交、AI输出监控、自动继续生成、内容复制到结果保存的全流程全自动，无需人工干预。
- 无缝整合用户现有剪贴板保存程序，实现一体化运行，保留其图形界面功能。
- 支持多种浏览器（Chrome、Edge）和多编码格式（UTF-8、ANSI），确保操作稳定、不重复、不丢失登录状态。
- 支持同时连接多个浏览器实例，实现多账号并行操作。

## Non-Goals (Out of Scope)
- 不支持非网页版AI平台（如API调用）。
- 不提供AI回复内容的分析或处理功能，仅负责保存原始回复。
- 不考虑人机验证处理（目前暂未遇到）。

## Background & Context
- 用户已拥有剪贴板识别保存程序，希望将该程序与网页自动化操作整合，实现全流程闭环。
- 手动向多个网页版AI提交问题存在效率低、重复劳动、易遗漏等问题。
- 网页AI平台通常需要登录状态，直接连接用户已打开的浏览器可以避免频繁验证和风控检测。

## Functional Requirements
- **FR-1**: 浏览器控制功能 - 支持连接已打开的Chrome和Edge浏览器，保留登录状态，确保连接稳定性，支持同时连接多个浏览器实例。
- **FR-2**: 标题/问题管理功能 - 支持从TXT文件读取或手动粘贴批量标题，自动去重，生成有序任务队列。
- **FR-3**: 网页AI自动化交互功能 - 自动定位输入框、发送按钮，监控AI输出状态，处理生成中断，完成后自动复制内容。
- **FR-4**: 一体化保存功能 - 整合用户现有剪贴板保存程序，保留其图形界面功能，自动读取剪贴板内容，匹配标题并保存到对应文件。
- **FR-5**: 无人值守与异常处理功能 - 全程自动执行任务，支持异常重试，记录错误日志，支持手动终止。
- **FR-6**: 图形用户界面 - 提供直观的图形界面，方便用户配置和监控任务执行。

## Non-Functional Requirements
- **NFR-1**: 稳定性 - 工具运行稳定，无崩溃、无卡死现象，连续运行4小时以上无异常。
- **NFR-2**: 易用性 - 配置简单，操作便捷，反馈清晰，用户可快速上手。
- **NFR-3**: 兼容性 - 适配Chrome、Edge最新版，自动识别TXT文件编码，适配不同网页AI平台。
- **NFR-4**: 安全性 - 不收集用户隐私信息，模拟真人操作，避免触发网页AI风控。
- **NFR-5**: 可扩展性 - 支持新增网页AI平台适配，支持自定义配置，预留扩展接口。

## Constraints
- **Technical**: Python 3.8及以上，依赖Playwright等库，需要开启浏览器远程调试模式。
- **Business**: 基础版本核心功能优先，后续可扩展其他功能。
- **Dependencies**: 依赖用户提供的完整、可运行的剪贴板保存程序源码。

## Assumptions
- 用户已安装并配置好Python环境。
- 用户已在浏览器中登录所需的网页AI平台。
- 用户提供的剪贴板保存程序源码功能正常，可读取剪贴板、匹配标题、保存文件。

## Acceptance Criteria

### AC-1: 浏览器连接
- **Given**: 用户已打开Chrome/Edge浏览器并登录网页AI平台
- **When**: 工具启动并连接浏览器
- **Then**: 成功连接浏览器，保留登录状态，无重新登录提示
- **Verification**: `programmatic`

### AC-2: 标题读取与去重
- **Given**: 存在包含重复标题的UTF-8或ANSI编码TXT文件
- **When**: 工具读取该文件
- **Then**: 成功读取文件，自动去重，生成有序任务队列
- **Verification**: `programmatic`

### AC-3: 自动化交互
- **Given**: 工具已连接浏览器并加载网页AI平台
- **When**: 工具执行任务队列
- **Then**: 自动定位输入框、发送按钮、继续生成按钮、复制按钮，操作准确，无遗漏
- **Verification**: `programmatic`

### AC-4: 输出监控
- **Given**: AI正在生成回复
- **When**: 工具监控输出状态
- **Then**: 准确识别生成中、中断、完成状态，生成中等待、中断时自动点击继续、完成后自动复制
- **Verification**: `programmatic`

### AC-5: 一体化保存
- **Given**: AI回复已复制到剪贴板
- **When**: 工具触发内嵌的剪贴板保存程序
- **Then**: 剪贴板内容读取正常，回复内容与标题精准匹配，成功保存到对应TXT文件，无丢失、无混淆
- **Verification**: `programmatic`

### AC-6: 无人值守
- **Given**: 工具启动并开始执行任务队列
- **When**: 用户离开电脑
- **Then**: 工具自动完成所有标题处理，无中途中断
- **Verification**: `human-judgment`

### AC-7: 异常处理
- **Given**: 遇到定位失败、发送失败等异常
- **When**: 工具执行任务
- **Then**: 自动重试，重试失败后跳过当前任务，继续执行下一个，记录错误日志
- **Verification**: `programmatic`
